# caloriesCalc
Простой калькулятор калорий, считает по стандартной формуле из интернета. Записывает в localStorage пол и активность.
Есть пол, возраст, вес, рост и 4 вида активности, соответственно получаем эти переменные, проверяем пол и активность в localStorage.
Функция initLocalSettings управляет классом активности для активности и пола.
Функция calcTotal считает по формуле (если какие-то данные не введены просто пишет ***).
Функция getStaticInformation получает ститическую инфу: активность, пол. Так же вызывает calcTotal (чтобы срабатывал при любом изменении).
Функция getDynamicInformation получает динамическую инфу: рост, вес, возраст. Так же стоит регулярка на проверку ввода только цифр(иначе делает красную окантовку). Свитчем 
проверяется в какой инпут производится ввод и соответственно записывается в нужную переенную результат, вызывается calcTotal.
Собрано с webpack
